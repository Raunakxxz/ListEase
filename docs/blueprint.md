# **App Name**: ListEase

## Core Features:

- Add Items: Allow users to quickly add new grocery items to their list with autocomplete suggestions.
- Mark as Purchased: Enable users to mark items as purchased with a simple tap or swipe.
- List Management: Enable users to create, name, and switch between multiple shopping lists.
- Item Reordering: Allow users to easily reorder items within a list by dragging and dropping them.
- Smart Suggestions: Suggest items to add based on past shopping habits and common grocery items using a tool to determine whether such a suggestion makes sense given past shopping data.

## Style Guidelines:

- Primary color: Soft pastel green (#B3E283) to evoke a sense of freshness and calm.
- Background color: Light beige (#F5F5DC) to provide a warm, neutral backdrop.
- Accent color: Muted yellow (#F0E68C) for interactive elements and highlights to subtly draw attention without overwhelming the user.
- Body and headline font: 'PT Sans', a humanist sans-serif to ensure the interface feels both modern and welcoming. Suitable for both headlines and body text.
- Use clean, minimalist icons in a pastel green to represent different grocery categories and actions.
- Employ a clean, card-based layout to clearly display list items and actions.
- Subtle animations and transitions when adding or marking items as purchased to provide positive feedback.
